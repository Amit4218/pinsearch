## Pinsearch

Pinsearch is an web application written in python fastapi, which provides fast and easy search access through the indian postal directory, and provide you with accurate information from the official dataset provided by the [OGD platform india](https://data.gov.in).

---

## pinsearch-sdk

Build your own pinsearch application with the lightweight python `pinsearch-sdk` [Learn More](https://github.com/Amit4218/pinsearch-sdk).

---

## pinsearch public api

Use the public api from pinsearch free of cost and no registration required. [Learn More](https://pinsearch.amit4218.fun/api-docs)

## Run locally

- **Step 1** : Clone the project

```bash
git clone https://github.com/Amit4218/pinsearch.git
cd pinsearch
```

- **Step 2** : Install depedency

```bash
uv sync
```

- **Step 3** : Run the application

```bash
uv run fastapi run # use dev for reload
```
